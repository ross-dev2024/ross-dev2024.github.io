2021-02-10

浏览器插件

  • Vue.js devtools 顾名思义，非常有用的浏览器 vue开发调试插件
  • React Developer Tools React 开发调试插件
  • Octotree - GitHub code tree 对github仓库的文件tree生成侧边栏，方便在线查阅浏览/跳转
  • XPath Helper 提取当前网站选中的 DOM 元素的 XPath，做埋点测试之类时有用

  • prompt 命令行 对话工具

  • execa shell 调用工具

  • chalk node终端输出美化工具

  • ora 终端 loading 工具

  • chokidar 文件监听工具

  • is-ci 检查当前环境是否是集成环境

  • Import Cost 显示包体积
